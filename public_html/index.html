About LinuxFAQ

LinuxFAQ is a community compilation of frequent questions and/or problems with Linux and their possible solutions. The goal of this project is to help guide new users in their endeavors.

Contributions can be made on GitHub.

Technical details

If you're reading this, you're probably wondering all the technical details of how LinuxFAQ works.

Here is a pretty straightforward explanation about everything:

  • Privacy LinuxFAQ will NOT accept or log any personal data directly. However, LinuxFAQ makes use of multiple services, such as: AWS (Server Hosting), GitHub (Repository Hosting), Cloudflare (Security), unpkg (CDN), jsDelivr (CDN), and Discord (Community Chat). Users may want to refer to each of these service's Terms of Service and/or Privacy Policy for more information.
  • Backend The backend (server side) is powered by Axum. It is hosted as a normal HTTP server, and secured with Cloudflare for HTTPS. The FAQ data is then cached from the repository.
  • Frontend The frontend (client side) uses XML HTTP requests to get updated information from the server.
